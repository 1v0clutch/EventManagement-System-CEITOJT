🐛 Dashboard Semester Bug Test

Bug Description

Issue: Schedule events (class schedules) were not showing on the calendar when viewing dates in a different semester than today.

Root Cause: The code was determining the semester based on TODAY's date, then checking if the SELECTED date fell within that semester. This prevented schedules from showing when browsing future/past semesters.

Fix: Changed logic to determine the semester based on the SELECTED date itself, then filter schedules that match that semester.
